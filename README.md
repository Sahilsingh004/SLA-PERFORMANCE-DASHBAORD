# SLA-PERFORMANCE-DASHBAORD
This project is an Excel-based SLA Performance &amp; Support Operations Dashboard built using PivotTables, PivotCharts, Slicers, and KPI cards. It helps track ticket volume, SLA compliance, resolution times, and channel/product performance.
